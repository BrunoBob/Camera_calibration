Detailed Description

Class that manage the graphical part of the sofware.

This class show the calibration picture and allow the user to select the four corners of the A4 calibration paper. Once the corners are selected it will display the border of the paper.

Handle function for a click event.

This function is called for each mouse event in the opencv window If the event is a left click it will select the position of the mouse as one of the corner of the calibration paper.
